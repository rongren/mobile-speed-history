# 🚴 모바일 속도계

자전거 주행 기록 앱. GPS로 속도/거리/시간을 측정하고 네이버 지도로 경로를 표시하며 주행 기록을 로컬 DB에 저장한다.

---

## 기술 스택

| 항목 | 내용 |
|---|---|
| Framework | Flutter |
| 지도 | flutter_naver_map |
| GPS | geolocator |

| 상태관리 | provider |
| 알림 | flutter_local_notifications |

## 화면 구성

바텀 네비게이션 5개로 구성

### 1. 속도계

- 자동차 속도계 스타일 CustomPainter 원형 게이지
- 상단 최고속도 범위 선택 (60 / 120 / 180 / 240 km/h)
- 속도에 따라 호 색상 변경 (파랑 → 초록 → 주황 → 빨강)
- 하단 통계 카드 (설정에서 표시 항목 선택 가능 — 거리/시간/최고속도/평균속도)
- 시작 / 정지 버튼
- 자동 일시정지 중 상태 표시 배너
- 시작 시 알림창에 측정 중 표시, 종료 시 알림 제거
- 백그라운드/잠금 상태에서도 시간 정확히 측정 (시작시간 기준 계산)
- 속도 보간 처리 (0.2초마다 부드럽게 변화)
- 주행 완료 팝업
  - 거리 / 시간 / 최고속도 / 칼로리(체중 설정 시) 또는 평균속도 표시
  - 메모 입력 (탭 → 바텀시트 입력창)

### 2. 지도

- 네이버 지도
- 앱 진입 시 현재 위치로 자동 이동 (애니메이션 없음)
- 내 위치 오버레이 표시
- 주행 중 실시간 경로 파란 선으로 표시
- 하단 거리 / 시간 / 최고속도 실시간 표시
- IndexedStack으로 탭 전환 시 상태 유지

### 3. 기록

탭 6개로 구성

#### 평균
- 1회 평균 (거리, 시간, 최고속도, 평균속도)
- 전체 누적 (총 횟수, 총 거리, 총 시간, 최고속도)

#### 연도별
- 막대 그래프 (거리/시간/최고속도/평균속도 선택)
- 막대 탭하면 해당 연도 상세 (요약 카드 + 월별 breakdown)
- 그래프 평균선 표시 (주황 점선)
- 다시 탭하면 상세 숨김

#### 월별
- 막대 그래프
- 막대 탭하면 해당 월 상세 (요약 카드 + 달력 히트맵 + 주차별 통계)
- 히트맵: 많이 탄 날일수록 진한 파란색

#### 일별
- 상단 기간 필터 (전체/7일/30일/90일/180일/365일)
- 안 탄 날도 그래프에 표시 (빈 막대)
- 막대 탭하면 해당 날 상세 (요약 카드 + 회차별 목록)
- 하단 요일별 평균 거리 막대 (접기/펼치기)

#### 상세
- 날짜 선택 (이전/다음 화살표, 셀렉트박스, 달력)
- 달력에 기록 있는 날 주황색 점 표시
- 해당 날짜 주행 목록 표시
- 카드 탭 → 상세 팝업 (통계 + 메모 수정 + 경로 보기 버튼)

#### 전체
- 연/월/일 필터 (우측 초기화 버튼)
- 정렬 기능 (날짜/거리/속도 오름차순·내림차순)
- 카드 탭 → 상세 팝업 (통계 + 메모 수정 + 경로 보기 버튼)
- 기록 삭제 기능 (- 버튼으로 삭제 준비, 음영 처리, 하단 삭제/취소 버튼)
- 최고 기록 뱃지 (최장거리/최고속도/최장시간)

### 4. 목표

미구현 (추후 월 목표 거리 설정 예정)

### 5. 설정

- 속도 단위 전환 (km/h ↔ mph)
- GPS 정확도 (고정밀 / 저전력)
- 최소 기록 거리 (기준 미만 주행은 저장 안 됨)
- 자동 일시정지 (저속 감지 시 타이머 일시정지)
- 기본 게이지 최고속도 (60 / 120 / 180 / 240 km/h)
- 체중 입력 (입력 시 주행 완료 팝업과 기록 화면에 칼로리 표시)
- 속도계 표시 항목 선택 (거리/시간/최고속도/평균속도 ON/OFF)

## 데이터 구조

```
ride_records
├── id            (자동증가)
├── year          (년)
├── month         (월)
├── day           (일)
├── totalDistance (km)
├── maxSpeed      (km/h)
├── avgSpeed      (km/h)
├── duration      (초)
├── pathPoints    (경로 JSON)
├── memo          (메모, 선택)
└── createdAt     (timestamp ms)
```
